## 🛠️ How to Install (ස්ථාපනය කරගන්නා ආකාරය)

1.  **Download Bot:** Mehi athi `TelegramTradeBot.ex5` file eka download karaganna.
2.  **Add to MT5:** MT5 hi `File -> Open Data Folder` wetha gos `MQL5/Experts` folder ekata meya copy karanna.
3.  **WebRequest Allow:** MT5 Settings hi (`Tools -> Options -> Expert Advisors`) pahatha URL eka **Allowed WebRequest** list ekata athulath karanna:
    * `https://script.google.com`
4.  **Activate License:** Bot eka chart ekata damu pasu diswana **Account ID** eka Admin wetha laba dee obage license eka active karaganna.
